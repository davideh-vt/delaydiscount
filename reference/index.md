# Package index

## Functions

- [`check_input_preconditions()`](https://davideh-vt.github.io/delaydiscount/reference/check_input_preconditions.md)
  : Check preconditions for an input data frame for the
  prepare_data_frame and jb_rule_check functions.
- [`dd_hyperbolic_model()`](https://davideh-vt.github.io/delaydiscount/reference/dd_hyperbolic_model.md)
  : This function gets model parameter estimates for each group,
  including hyperparameters and estimates of the variance components
  (the observation error and subject random effect). It also gets F-test
  results for pairwise equality for the hyperparameter for each pair of
  groups, and overall equality of all hyperparameters.
- [`estimate_hyperbolic_model_params()`](https://davideh-vt.github.io/delaydiscount/reference/estimate_hyperbolic_model_params.md)
  : Estimate the hyperparameters for each group, as well as the variance
  components of the model, using maximum likelihood estimation.
- [`get_subj_est_ln_k()`](https://davideh-vt.github.io/delaydiscount/reference/get_subj_est_ln_k.md)
  : Return ln(k) estimates for each subject
- [`hyperbolic_model_f_test()`](https://davideh-vt.github.io/delaydiscount/reference/hyperbolic_model_f_test.md)
  : Perform an F-test for equality of some combination(s) of
  hyperparameters.
- [`jb_rule_check()`](https://davideh-vt.github.io/delaydiscount/reference/jb_rule_check.md)
  : Apply rule checks to each subject.
- [`prepare_data_frame()`](https://davideh-vt.github.io/delaydiscount/reference/prepare_data_frame.md)
  : Prepare a data frame containing discounting for use on the other
  functions in this package.

## Datasets

- [`remedi`](https://davideh-vt.github.io/delaydiscount/reference/remedi.md)
  : REMEDI data
